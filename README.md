Очень простая реализация трансформер-модели для генерации текста и ведения диалога на основе собственного корпуса сообщений.

Модель обучается на текстах из Excel-файла (result.xlsx) или из книги и после обучения может поддерживать чат с пользователем.

Композиция:

custom_tokens.py — функции для токенизации, обработки редких слов и маскирования токенов.

indexing.py — кодировка токенов в индексы.

model.py — архитектура Transformer.

main.py — обучение модели и запуск чата.

result.xlsx — входные данные.

transformer_checkpoint.pth — файл сохранения модели после обучения.
